# Sum from 1 to N in C

Simple program written in C that reads a positive integer and:

* Prints all numbers from 1 up to the given number
* Calculates and displays the sum of these numbers using a loop

## Example

Input:

```
5
```

Output:

```
Numbers: 1
Numbers: 2
Numbers: 3
Numbers: 4
Numbers: 5

The sum of the numbers is: 15
```

## How to run

Compile:

```
gcc program.c -o program
```

Run:

```
./program
```
